# PDF Font Sources

Predicta report PDFs bundle these fonts so generated artifacts do not depend on
local machine font availability.

- Cormorant Garamond SemiBold/Bold: Google Fonts, SIL Open Font License 1.1.
- Source Serif 4 Regular/SemiBold/Bold: Google Fonts, SIL Open Font License 1.1.
- Noto Serif Devanagari Regular: The Noto Project, SIL Open Font License 1.1.
- Noto Serif Gujarati Regular: The Noto Project, SIL Open Font License 1.1.

The license text is included in `OFL.txt`.
