# Predicta Intelligence Phase 0 Broken-Record Phrase Ledger

Phase keyword: `PREDICTA_INTELLIGENCE_PHASE_0_CURRENT_CHAT_REDLINE_AND_PERSONA_LOCK`
Status: `LOCKED`
Date: 2026-06-15

This ledger records phrase patterns that must be removed, suppressed, or
demoted in future phases.

## Hard No-Go Phrases

These must not appear as primary user-facing answer patterns:

- `This house represents...`
- `This chart governs...`
- `This section helps you understand...`
- `KP uses...` before the answer.
- `Jaimini uses...` before the answer.
- `Numerology shows how the name projects...` before the answer.
- `Signature analysis is reflective guidance...` as the opening line when
  confirmed traits exist.
- `Provider decision: local_memory_answer. No AI credit is needed.` repeated in
  normal user answers.
- `Kundli Karma local memory is ready.` as the first line of a personal reading.
- `Ask like...` as the main response when the user asked for help.
- `Press Ask, or type your follow-up.` as a repeated landing line.
- `The answer will now stay grounded in...` as repeated room-intro copy.
- `Use this report as a real starting point, not as a teaser.`
- `This report helps you understand...`
- `Classical proof` as a user-first Life Atlas/Numerology/Signature concept.
- `Fun chart note` inside serious reports or report chats.

## Suspicious Patterns

These can be useful in audits or appendices, but should not dominate chat:

- `Chart evidence` as a mandatory section in every answer.
- `Report memory enforcement`.
- `Room contract enforcement`.
- `Discipline handoff enforcement`.
- `Calculation-engine reply:` on every deterministic answer.
- `No AI credit is needed` repeated after every local action.
- `Premium adds...` before free value is delivered.

## Correct Replacement Style

| Bad Pattern | Replacement |
|---|---|
| `D10 governs career.` | `Career grows through structured responsibility, but recognition comes slowly unless timing support strengthens.` |
| `KP needs a specific question.` | `I can start with career readiness now. For a final verdict, choose job change, promotion, foreign transfer, or write your exact question.` |
| `This section includes Mahadasha, Antardasha...` | `This life period is asking you to stabilize work, money, and responsibility before taking a bigger leap.` |
| `Provider decision: local_memory_answer.` | `I can answer this from your calculated Predicta memory without using an AI question.` |
| `Signature guidance is reflective...` | `Your confirmed signature traits point to a cautious but improving self-presentation rhythm. Treat this as reflection, not proof.` |

## Phase 1 Test Requirement

Phase 1 must add a no-schooling phrase gate that rejects these phrases when they
appear before the direct answer.
