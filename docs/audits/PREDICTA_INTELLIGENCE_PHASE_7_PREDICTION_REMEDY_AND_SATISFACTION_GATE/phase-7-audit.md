## Scope Verified

- Predicta answers now expose a satisfaction frame across the major prediction modes:
  `Direct answer`, `Timing`, `Confidence`, `Action/remedy`, and `Evidence`.
- Main Predicta event answers use the Event Oracle reply contract with direct answer, timing window, likely trigger, confidence, action/remedy, evidence, conflicts, and next action.

- Money/property decisions that route through deterministic `decision-timing` now still satisfy the prediction/remedy/evidence contract.
