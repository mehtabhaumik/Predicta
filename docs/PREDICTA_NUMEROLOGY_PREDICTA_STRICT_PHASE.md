# Predicta Numerology Predicta Strict Phase

This document defines the strict rebuild phase for Numerology Predicta across
web, mobile, chat, reports, PDF output, and Predicta memory.

Numerology Predicta must stop feeling like four static number cards. It must
feel like a polished, living number map of the person: identity, name rhythm,
birth code, timing cycle, strengths, cautions, and premium name/compatibility
insight.

## Phase Name

`PREDICTA_NUMEROLOGY_PREDICTA_PHASE_1_NUMBER_IDENTITY_DASHBOARD_AND_DOSSIER_REBUILD`

## Current Baseline Problems

The current implementation is a good foundation, but it is not premium enough:

- Shared logic already calculates `nameNumber`, `birthNumber`, `destinyNumber`,
  `personalYear`, `personalMonth`, and `personalDay`.
- Web currently shows a profile summary and four simple cards.
- Mobile currently shows an even simpler four-card profile and chat CTA.
- PDF currently has one useful Numerology section, but not a premium dossier.
- Numerology does not yet have a strong wow visual system.
- Numerology does not yet show a living yearly/monthly rhythm map.
- Numerology does not yet show polished missing/repeated number patterns.
- Numerology does not yet make name refinement, compatibility, or brand/baby
  name comparison feel like premium value.

## Non-Negotiable Rules

1. Numerology Predicta is its own school/world.
2. Numerology reports must not silently mix Vedic, KP, Nadi, or Signature logic.
3. Vedic/KP/Nadi/Signature reports must not silently include Numerology sections
   unless an approved synthesis report explicitly says so.
4. Numerology Predicta may participate in `Predicta Life Atlas` only as a
   clearly labeled synthesis evidence layer.
5. Free Numerology must be useful and complete enough to build trust.
6. Premium/paid Numerology must add depth, name refinement, compatibility,
   timing, and interpretation quality, not fear or fake certainty.
7. Do not say a number guarantees success, wealth, marriage, fame, failure,
   illness, or destiny.
8. Do not pressure users to change names through fear.
9. Do not say `change your name or your life will fail`.
10. Do not call a name `bad`, `cursed`, or `doomed`.
11. Use reflective, practical language:
    - `may support`
    - `can reflect`
    - `this name rhythm leans toward`
    - `this cycle is better used for`
    - `this number pattern asks for`
12. Every calculated number must expose method/evidence in a user-friendly way.
13. Missing data must show an honest pending state, not fabricated numerology.
14. Web, mobile, reports, and Predicta chat must use the same shared Numerology
    intelligence contract.
15. Any name comparison or compatibility score must show confidence and
    limitations.

## Wow Moments

Numerology must feel alive, visual, and personal.

### Personal Number Mandala

Show a premium radial mandala with:

- Name number
- Birth number
- Destiny/life-path number
- Personal year
- Personal month
- Personal day

Each number must have:

- color token
- keyword
- short meaning
- accessible label

The mandala must appear on:

- web Numerology section
- mobile Numerology section
- Numerology PDF opening spread

### Name Energy Scanner

Animate the user's name resolving into number values:

```text
B H A U M I K -> 2 + 5 + 1 ... -> 31/4
```

Requirements:

- Use the selected name method.
- Show compound number and root number.
- Keep animation tasteful and readable.
- Respect reduced-motion preferences.
- Premium may compare name variations, brand names, baby names, spouse names, or
  business names.

### Personal Year Timeline

Show the current year as a 12-month number rhythm.

Each month should include:

- month label
- cycle number
- short action keyword, such as `Build`, `Express`, `Simplify`, `Launch`,
  `Learn`, `Repair`, `Close`, or `Wait`
- one-line guidance

### Missing And Repeated Number Grid

Add a polished 1-9 grid showing:

- strong numbers
- repeated numbers
- missing numbers
- balanced numbers

This may use a Lo Shu-style grid or a number frequency map, but the method must
be documented and not mixed with unsupported claims.

### Life Theme Sentence

Generate one powerful distilled line from the user's number pattern.

Example:

```text
Your life number pattern asks you to turn creative intensity into disciplined service.
```

This sentence must appear in:

- Numerology section hero
- Numerology report summary
- Numerology PDF opening spread

### Name Fit Score

Premium-only, carefully framed.

Example:

```text
Current name alignment: 78/100
```

Breakdown:

- expression
- stability
- public rhythm
- destiny support

Rules:

- never call a name bad or doomed
- show confidence and limitations
- explain what would improve the score without fear

### Best Use Of This Cycle

For personal year/month/day, show a practical action label:

- `Start`
- `Repair`
- `Wait`
- `Launch`
- `Learn`
- `Close`
- `Build`
- `Simplify`

The section must tell the user what to lean into now and what to avoid.

### Numerology Compatibility Lens

Premium-only.

It may compare:

- partner
- business partner
- brand name
- baby name
- public name
- company/product name

Required outputs:

- where rhythms support
- where timing clashes
- how to work better together
- confidence
- limitations

### Lucky/Supportive Toolkit

Use carefully and without gimmicks.

May include:

- supportive colors
- supportive days
- supportive numbers
- affirmations
- habits
- gentle remedies or alignment practices

Rules:

- frame as alignment aids
- do not promise magic results
- do not make fear-based recommendations

`Your Number Signature`

It must include:

- large personal number mandala
- life theme sentence

- top gifts
- top caution pattern
- clean evidence note

Free gets a concise version. Premium/paid gets the full premium visual and

## Report And PDF Guardrails

Do not write:

- `This number guarantees success.`
- `Change your name or your life will fail.`
- `This is your fixed destiny.`
- `This name is bad.`
- `This number proves you will become rich.`

Use instead:

- `This number pattern may support...`
- `This name rhythm can reflect...`
- `This cycle is better used for...`
- `This name spelling may feel more aligned with...`
- `This pattern is worth using as reflection, not as fate.`

Reports must include:

- method note
- confidence/limitations
- no guaranteed outcomes
- no fear-based name-change language
- no hidden mixing with Vedic/KP/Nadi/Signature

Predicta must know:

- Numerology Predicta is a separate school/world.
